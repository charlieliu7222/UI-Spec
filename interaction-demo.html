🎯 互動功能演示頁面

這個頁面展示了如何在 UI Spec Builder 中使用互動功能標記，以及這些功能在實際頁面中的運作方式。

📝 範例 1: 按鈕點擊事件

📋 UI 規格：
├─ Button: "點我試試" (primary, onClick: handleClick)

功能標記：

onClick: handleClick

✏️ 範例 2: 輸入框事件

📋 UI 規格：
├─ Input Field (placeholder: "輸入你的名字", onChange: handleChange, onFocus: handleFocus, onBlur: handleBlur)

功能標記：

onChange: handleChange onFocus: handleFocus onBlur: handleBlur

📤 範例 3: 表單提交事件

📋 UI 規格：
├─ Form (max-width: 400px, onSubmit: handleSubmit)
  ├─ Input Field (placeholder: "Email", type: email, onChange: handleEmailChange)
  ├─ Button: "提交" (primary, onClick: handleSubmit)

功能標記：

onSubmit: handleSubmit onChange: handleEmailChange onClick: handleSubmit

💡 如何在 UI Spec Builder 中使用

  1. 在右側結構中點擊要添加功能的組件行
  2. 在左側「⚡ 互動功能」面板中選擇功能類型
  3. 系統會自動添加功能標記到該行
  4. 點擊「產生規格」查看完整文檔
  5. 將規格分享給開發團隊實現
